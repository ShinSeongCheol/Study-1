# 8.09

03. HTML 기본 문서 만들기
```
HTML의 기본 구조와 4가지 태그
HTML 파일 만들기, 라이브 서버 설치
기본 구조 입력방법(! 입력 후 tab/Enter)
웹 문서 구조를 만드는 시맨틱 태그
header - 헤더 영역
nav - 다른 위치로 연결, 다른 웹문서로 연결하는 링크 생성
main - 핵심이 되는 내용 삽입
section - 콘텐츠 영역
aside - 본문 외 왼/오른쪽, 아래쪽에 사이드 바 생성
footer - 맨 아래쪽 사이트제작정보, 저작권정보, 연락처 등 삽입
div - id나 class 속성을 사용해 문서구조 생성이나 스타일 적용에 사용
```
04.  웹 문서에 다양한 내용 입력하기
```
태그
<h1~6> - 제목
<p> - 텍스트 단락
<br> - 줄바꿈
<blockquote> - 인용문
<strong>, <b> - 텍스트 굵게
<em>, <i> - 텍스트 기울이기

목록
<ol>, <li> - 순서 있는 목록
<ul>, <li> - 순서 없는 목록
<dl>, <dt>, <dd> - 설명 목록

표
<table>, <caption> - 표 생성
<tr> - 행 생성
<td>, <th> - 셀 생성
<thead>, <tbody>, <tfoot> - 표의 구조 지정
rowspan, colspan 속성 - 행,열 병합
<col>, <colgroup> - 열을 묶어줌

이미지
<img> - 이미지 삽입

오디오,비디오
<object>, <embed> - 다양한 멀티미디어 파일 삽입
<audio>, <video> - 오디오, 비디오파일 삽입

하이퍼링크
<a> - 링크 삽입, href 속성으로 링크할 주소 작성
```
